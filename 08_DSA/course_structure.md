## **2. Lesson Breakdown**

### **Lesson 1: Binary Search, Linked Lists, and Complexity**
- **Subtopics:**
  - Problem solving methodology for coding problems
  - Linear vs. Binary Search (with code)
  - Linked Lists basics
  - Big O notation & time/space complexity
  - Code testing strategy with test cases
  - Python functions and good naming practices
- **Assignments:**
  - [Binary Search Practice (Assignment 1)](https://jovian.ai/aakashns/python-binary-search-assignment)
  - **Practice Tasks:**
    - Linear Search, Binary Search notebook implementations
    - Systematic problem solving using the provided template

***

### **Lesson 2: Binary Search Trees, Traversals, and Recursion**
- **Subtopics:**
  - Binary Trees and Binary Search Tree (BST) concepts
  - BST traversals (in-order, pre-order, post-order)
  - Recursive thinking in programming
  - Self-balancing trees – AVL Trees (introduction)
  - Recursion basics
- **Assignments:**
  - [Hash Tables & Python Dictionaries (Assignment 2)](https://jovian.ai/aakashns/python-hash-tables-assignment)
  - **Practice Tasks:**
    - Binary Tree and BST implementation
    - Traversal algorithms

***

### **Lesson 3: Sorting Algorithms and Divide & Conquer**
- **Subtopics:**
  - Divide and Conquer paradigm
  - Popular sorting algorithms (Merge Sort, Quick Sort, etc.)
  - Writing robust sorting code
  - Analyzing and comparing sort complexity
- **Assignments:**
  - [Divide and Conquer Practice (Assignment 3)](https://jovian.ai/aakashns/python-divide-and-conquer-assignment)
  - **Practice Tasks:**
    - Implementing Merge Sort
    - Using comparison functions for custom sorts

***

### **Lesson 4: Recursion and Dynamic Programming**
- **Subtopics:**
  - Recursion – methods, intuition, and code structure
  - Memoization for optimized recursion
  - Dynamic Programming (DP): strategy, applications
  - Example DP problems (like Knapsack)
- **Assignments:**
  - Dynamic Programming practice problems notebook
  - **Practice Tasks:**
    - Recursive and DP solutions for real interview-style problems

***

### **Lesson 5: Graph Algorithms**
- **Subtopics:**
  - Graph data structures (representation in Python)
  - Breadth-First Search (BFS)
  - Depth-First Search (DFS)
  - Shortest paths in graphs
  - Applications and common interview questions
- **Assignments:**
  - [Graphs and Graph Algorithms notebook](https://jovian.ai/aakashns/python-graph-algorithms) (starter)
  - **Practice Tasks:**
    - Implement BFS and DFS algorithms
    - Shortest path exercises
